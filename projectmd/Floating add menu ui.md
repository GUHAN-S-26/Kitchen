# Open Animation Sequence

1. Center button glows
2. Wave path draws upward
3. Floating dots animate
4. Action buttons appear one by one
5. Labels fade in

Duration:
250ms – 400ms

# Close Animation Sequence

1. Labels fade out
2. Action buttons collapse
3. Wave line shrinks
4. Center button rotates back to +
